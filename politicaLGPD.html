Política LGPD

O Hospital Promater está comprometido com a conformidade com a Lei Geral de Proteção de Dados (LGPD). Esta política explica como tratamos os dados pessoais em conformidade com a LGPD.

Coleta de Dados

Coletamos dados pessoais necessários para fornecer nossos serviços e garantir a segurança dos nossos pacientes. Os dados são coletados de forma transparente e com a devida autorização.

Uso de Dados

Os dados coletados são usados para fins específicos, como a gestão de consultas e o atendimento aos pacientes. Não utilizamos dados pessoais para fins não autorizados.

Direitos dos Titulares

Você tem o direito de acessar, corrigir e excluir seus dados pessoais. Para exercer seus direitos, entre em contato conosco através dos canais disponibilizados.

Segurança dos Dados

Adotamos medidas técnicas e administrativas para proteger seus dados contra acesso não autorizado, perda ou divulgação indevida.

Alterações na Política

Podemos atualizar nossa política de LGPD periodicamente. Recomendamos que você revise esta página regularmente para estar informado sobre quaisquer mudanças.
